## ملاحظات مهمة

1. **نظام HSL**: جميع الألوان تستخدم نظام HSL (Hue, Saturation, Lightness)
2. **CSS Variables**: الألوان محفوظة كمتغيرات CSS في `:root` و `.dark`
3. **Tailwind Integration**: الألوان متكاملة مع Tailwind CSS
4. **Theme Support**: الموقع يدعم الوضع الفاتح (Light) والداكن (Dark)
5. **Responsive**: الألوان تعمل على جميع أحجام الشاشات

## كيفية الاستخدام

### في CSS
```css
background-color: hsl(var(--primary));
color: hsl(var(--primary-foreground));
```

### في Tailwind
```jsx
<div className="bg-primary text-primary-foreground">
  Content
</div>
```

### في JavaScript/TypeScript
```typescript
const primaryColor = getComputedStyle(document.documentElement)
  .getPropertyValue('--primary');
```
